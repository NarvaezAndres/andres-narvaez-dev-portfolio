# 📂 Proyectos del Portafolio

En esta sección se encuentran los principales proyectos que desarrollo como parte de mi portafolio profesional.

### 🔹 Sistema de Gestión de Inventario

Sistema backend para administrar inventarios y control de productos.

Características:

* Gestión de productos
* Control de stock
* API REST

---

### 🔹 API de Autenticación

Sistema de autenticación para manejo de usuarios.

Características:

* Inicio de sesión seguro
* Autenticación con JWT
* Control de acceso por roles

### 🔹 API de Gestión de Tareas

API REST para la administración de tareas.

Características:

* Operaciones CRUD
* Integración con base de datos
* Estructura basada en buenas prácticas
